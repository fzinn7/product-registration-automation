# 🚀 Automação de Cadastro de Produtos

Este projeto automatiza o processo de cadastro de produtos em um sistema web utilizando Python.

## 🧠 Objetivo

O objetivo é simular um fluxo real de automação de tarefas repetitivas, como:
- Acessar um sistema
- Realizar login
- Ler uma base de dados
- Cadastrar produtos automaticamente

---

## 🛠️ Tecnologias utilizadas

- Python
- PyAutoGUI
- Pandas

---

## ⚙️ Funcionalidades

✅ Abre o navegador automaticamente  
✅ Acessa o sistema web  
✅ Realiza login  
✅ Lê um arquivo CSV com produtos  
✅ Cadastra produtos automaticamente  

---

## ⚠️ Observações importantes

Este projeto utiliza automação baseada em coordenadas de tela (PyAutoGUI).  
Por isso, pode haver variações dependendo da resolução do monitor ou posição da janela.
